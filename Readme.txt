# Suivi des ventes – Power BI

## Description du projet
Ce projet a pour objectif de mettre en place un pipeline complet d'automatisation des ventes, depuis la collecte des données jusqu'à la visualisation sur un tableau de bord interactif Power BI.  
Il permet de suivre les ventes réalisées par chaque commercial, d'analyser les performances des produits et de générer des indicateurs clés automatiquement.


## Fonctionnalités
- Collecte automatique des données via Google Forms → Google Sheets  
- Nettoyage et transformation des données avec Power Query (gestion des séparateurs décimaux, suppression des doublons, création de colonnes calculées)  
- Tableau de bord Power BI   
  - Chiffre d’affaires total  
  - Nombre de ventes  
  - Top produits  
  - Évolution mensuelle du CA  
- Automatisation   
  - Rafraîchissement automatique des données depuis Google Sheets  
  - Mesures DAX pour calculer les montants totaux et indicateurs temporels
